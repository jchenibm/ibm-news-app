# ibm-news-app

A simple Vue & Flask App to load and read IBM What's New articles.

![screenshot](public/screenshot.png)

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Run back-end server
```
python server.py
```
